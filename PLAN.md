# LFP-2 운영 단위 선택 진입 화면 계획

## 범위

- 첫 로그인 화면에서 활성 운영 단위(기수/커리큘럼)를 먼저 선택하게 한다.
- 기수 선택은 드롭다운으로 제공한다.
- 기수 입장 코드와 전체 관리자 코드를 서로 다른 form으로 분리한다.
- 로그인 실패 시 선택한 운영 단위를 유지한 채 오류를 표시한다.
- 전체 관리자 로그인 실패 시 관리자 입력 영역을 열어 오류를 표시한다.
- DB 목록 로드 실패 시 기본 운영 단위(`3기`)로 fallback 한다.

## 변경 파일

| 파일 | 변경 |
|------|------|
| `src/app/meetup-dashboard.tsx` | 비인증 첫 화면에 운영 단위 드롭다운과 전체 관리자 별도 입력 영역 추가 |
| `src/app/actions.ts` | 로그인 실패 시 기수/관리자 오류 query를 분리 |
